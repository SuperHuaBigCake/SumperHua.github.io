Attack Tools

常用工具使用以及下载地址

Afrog

漏洞扫描工具

https://github.com/zan8in/afrog

  • 批量检测目标
afrog.exe -T urls.txt
  • 检测单个目标
afrog.exe -t https://example.com
  • 更新
afrog.exe -un

Nuclei

漏洞扫描工具

https://github.com/projectdiscovery/nuclei

  • 使用社区提供的模块扫描单个目标
nuclei.exe -u https://example.com
  • 使用社区提供的模块扫描多个目标
nuclei.exe -list urls.txt
  • 扫描指定Yaml格式POC
nuclei.exe -u https://example.com -t kingdee-erp-deserialization.yaml
  • 更新模板
nuclei.exe -update-templates
  • 更新Nuclei
nuclei.exe -update
  • 生成扫描报告
nuclei.exe -u https://example.com -o result.txt

Xpoc

漏洞扫描工具

https://github.com/chaitin/xpoc

  • 扫描指定目标i
xpoc.exe -t https://example.com -o result.html
  • 查看所有云端POC
xpoc.exe list -a
  • 批量扫描
xpoc.exe -i urls.txt
  • 同步最新插件
xpoc.exe pull
  • 更新Xpoc
xpoc.exe upgrade

SpringBootScan

针对SpringBoot框架的扫描工具

https://github.com/AabyssZG/SpringBoot-Scan

  • 安装python依赖库
python3 -m pip install -r requirements.txt
  • 对单一目标进行信息泄露扫描
python3 SpringBoot-Scan.py -u example.com
  • 对批量目标进行信息泄露扫描
python3 SpringBoot-Scan.py -f url.txt
  • 对单一目标进行漏洞利用
python3 SpringBoot-Scan.py -v https://example.com
  • 扫描并下载SpringBoot敏感文件
python3 SpringBoot-Scan.py -d example.com

Packer-Fuzzer

针对使用Packer组件的扫描工具

https://github.com/rtcatc/Packer-Fuzzer

  • 安装python依赖库
python3 -m pip install -r requirements.txt
  • 扫描单个网站Packer泄露接口漏洞
python3 PackerFuzzer.py -u PackerFuzzer.py
  • 忽略HTTPS证书错误
python3 PackerFuzzer.py -u PackerFuzzer.py -f 1

Dirsearch

目录扫描工具

https://github.com/maurosoria/dirsearch

  • 安装python依赖库
python3 -m pip install --upgrade pip

python3 -m pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
  • 对网站进行基础目录扫描
python3 dirsearch.py -u https://target
  • 只扫描网站php、html、js文件
python3 dirsearch.py -e php,html,js -u https://target
  • 指定字典进行目录扫描
python3 dirsearch.py -u https://target -w SpringBootAPI.txt
  • 对目录进行递归扫描
python3 dirsearch.py -u https://target -r
  • 只显示200、302状态码的请求
python3 dirsearch.py -u https://target -i 200,302
  • 不显示404、403状态码的请求
python3 dirsearch.py -u https://target -x 404,403
  • 设置代理
python3 dirsearch.py -u https://target --proxy socks5://10.10.0.1:8080

python3 dirsearch.py -u https://target --proxy 127.0.0.1:8080

python3 dirsearch.py -u https://target --proxylist proxyservers.txt

Netspy

探测内网可达网段工具

https://github.com/shmilylty/netspy

  • 使用icmp进行探测网段
netspy.exe is
  • 使用tcp进行探测网段
netspy.exe ts -p 22 -p 3389

注：如果不指定-p参数，netspy默认探测21, 22, 23, 80, 135, 139, 443, 445, 3389, 8080端口

  • 使用udp进行探测网段
netspy.exe us -p 53 -p 137

注：如果不指定-p参数，netspy默认探测53, 123, 137, 161, 520, 523, 1645, 1701, 1900, 5353端口

  • 使用icmp强制进行段内所有IP存活探测
netspy.exe -c 192.168.91.0/24 -r 255 -f is
  • 使用icmp急速探测
 netspy.exe -x is

注：急速模式协程数量为cpu核数*40，只探测段内网关

Fscan

漏洞扫描工具

https://github.com/shadow1ng/fscan

  • 基础使用
fscan.exe -h 192.168.0.1/24
  • 跳过存活检测
fscan.exe -h 192.168.0.1/24 -np
  • 线程为100扫描，默认为600
fscan.exe -h 192.168.0.1/24 -t 100
  • 跳过web poc扫描
fscan.exe -h 192.168.0.1/24 -nopoc
  • 跳过sql、ftp、ssh等密码的爆破
fscan.exe -h 192.168.0.1/24 -nobr
  • 设置扫描的端口
fscan.exe -h 192.168.0.1/24 -p 1-65535

OneForAll

子域名收集工具

https://github.com/shmilylty/OneForAll

  • 安装python依赖库
python3 -m pip install -U pip setuptools wheel -i https://mirrors.aliyun.com/pypi/simple/

python3 -m pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
  • 扫描单个域名的子域名
python3 oneforall.py --target example.com run

OneForAll在默认参数正常执行完毕会在results目录生成相应结果

  • 扫描txt文档中域名的子域名
python3 oneforall.py --targets ./example.txt run

ksubdomain

无状态子域名爆破工具

https://github.com/knownsec/ksubdomain

  • 使用内置字典爆破
ksubdomain.exe -d seebug.org
  • 使用字典爆破域名
ksubdomain.exe -d seebug.org -f subdomains.txt
  • 爆破三级域名
ksubdomain.exe -d seebug.org -l 2
  • 使用网络API接口获取域名
ksubdomain.exe -d seebug.org -api
  • 完整模式，先使用网络API，在此基础上使用内置字典爆破
ksubdomain.exe -d seebug.org -full
